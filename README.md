# ☕️ Readly (阅读搭子)

> 一款充满极客精神与产品温度的「沉浸式 AI 阅读伴侣」。
> 拒绝平庸的机器翻译，以资深产品经理的视角，为你精准提炼外文博客与长文的核心价值。

## ✨ 核心理念 (Philosophy)
- **One Card = One Insight**：打破粗放的段落归纳，实行“一个卡片只讲一个锐利亮点”的降维打击。
- **精准溯源**：卡片底部的 `[定位到原文]` 按钮，百发百中地跳到最关键的那句原话，绝不迷路。
- **极致留白**：深摩卡与浅焦糖的温润纸质配色，自适应长短句排版，誓死捍卫极简审美。
- **知识闭环**：支持鼠标划线提取 `[YOUR TAKEAWAY]`，与 AI 洞察在同一时间线交织沉淀。

## 📦 如何安装使用？(Installation)

由于本项目采用完全免费的 **BYOK (Bring Your Own Key)** 模式，未上架 Chrome 商店，你需要通过“开发者模式”进行安装。

### 1. 下载并安装插件
1. 前往 GitHub 的 Releases 页面下载最新的 `.zip` 压缩包（或者直接下载本项目根目录下的 `build/chrome-mv3-prod/` 文件夹）。
2. 将压缩包解压到一个固定的文件夹中。
3. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展程序管理页面。
4. 打开右上角的 **“开发者模式”** 开关。
5. 点击左上角的 **“加载已解压的扩展程序”**，选择你刚刚解压的那个文件夹。
6. 成功！你现在可以在 Chrome 右上角的扩展图标里找到咖啡色的 Readly 了。

### 2. 配置大模型 API Key
为了让你的阅读搭子开始工作，你需要自备大模型钥匙：
1. 在 Chrome 右上角右键点击 Readly 的咖啡色图标，选择 **“选项 (Options)”**。
2. 在弹出的设置页面中填入信息：
   - **模型名称**：
     - 如果你使用官方接口，填写 `deepseek-chat`
     - 如果你使用火山引擎（字节云），填写对应的推理接入点，如 `ep-20241234-xxx`
   - **API Key**：填入你申请的有效密钥。
3. 点击“保存配置”，你的数据仅保存在浏览器本地，绝对安全。

### 3. 开始沉浸阅读
1. 随意打开一篇深度的长文博客（如 GatesNotes, Medium 等）。
2. 点击右上角的 Readly 扩展图标，唤出侧边栏。
3. 点击 **“☕️ 开启阅读时光”**。
4. 喝口咖啡，享受高维度的洞察提炼。
5. **划线记录**：在阅读过程中，选中任何一段有共鸣的文字，点击悬浮的 `[提取]` 图标，写下你的专属 Takeaway。

## 🛠 技术栈
- [Plasmo](https://docs.plasmo.com/) (Browser Extension Framework)
- React 18 + TailwindCSS
- `@mozilla/readability` (Core Extraction Engine)
- DeepSeek V3 (LLM)

## 🔒 隐私声明
Readly 坚持“完全本地化”的隐私原则：
1. 插件仅通过客户端 JS 提取你当前正在阅读的网页正文。
2. 正文数据直接发送至你自行配置的第三方 LLM 接口，不会经过任何其他服务器。
3. 你的 API Key、历史总结与划线记录，全部存储在 `chrome.storage.local` 本地缓存中。
4. 我们不收集、不上传、不分析任何用户的浏览行为。
